## 📋 وصف المشروع

هذا المصنع يبني نظام لايف ISO كامل على GitHub Actions اسمه **KSO-HANI V34.FINAL11** مبني على Ubuntu 24.04 Noble بواجهة i3.

|---------|---------|
| **الاسم** | KSO-HANI V34.FINAL11 [KSO] |
| **الأساس** | Ubuntu 24.04 Noble |
| **المعمارية** | amd64 |
| **الواجهة** | i3 window manager |

## 🚀 طريقة الاستخدام

### الطريقة الأولى: تشغيل main.py محلياً ثم رفع على GitHub

**المتطلبات:** Python 3 فقط

```bash
# تشغيل السكريبت لإنشاء ملفات GitHub Actions
python3 main.py
```

سيقوم السكريبت بإنشاء:
```
.github/
└── workflows/
    └── build.yml
```

ثم:
1. نزّل المشروع كـ ZIP من GitHub أو ارفع الملفات
2. اذهب إلى **GitHub > Actions**
3. اختر **KSO-HANI V34.FINAL11 - ISO Factory [KSO]**
4. اضغط **Run workflow**
5. انتظر اكتمال البناء (حوالي 30-60 دقيقة)
6. نزّل الـ ISO من قسم **Artifacts**

---

### الطريقة الثانية: استخدام ملف build.yml مباشرة

إذا كان الملف `.github/workflows/build.yml` موجوداً بالفعل:

1. اذهب إلى **GitHub > Actions**
2. اختر الـ workflow
3. اضغط **Run workflow**
